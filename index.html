Quick start
Audit mode — readiness check, no changes
.\Invoke-CISHardening.ps1 -AuditMode
Preview all changes (WhatIf)
.\Invoke-CISHardening.ps1 -WhatIf
Apply Level 1 hardening
.\Invoke-CISHardening.ps1 -Level L1
Apply specific modules only
.\Invoke-CISHardening.ps1 -Level L1 -Modules AccountPolicy,AuditPolicy,TLSHardening
Apply Level 2 with ASR in Block mode
.\Invoke-CISHardening.ps1 -Level L2 -ASRMode Block
Rollback all changes
.\Invoke-CISHardening.ps1 -Rollback -RollbackPath .\CISHardening_Output\Rollback\<timestamp>
